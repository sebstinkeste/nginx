# Pull and Run image Nginx
```
$ docker pull sebstinkeste/nginx
```


```
$ docker run -it sebstinkeste/nginx bash

```
# Presentation
The default workdir is /var/www/ and the default Nginx Document Root is /usr/share/nginx/html.
